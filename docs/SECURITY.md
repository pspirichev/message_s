## Current Security Status

Disclaimer: At this stage, the project is not yet fully functional. If you are reading this and the project appears to be working, please be advised that it is still in an experimental phase.

I do not assume any responsibility for vulnerabilities, data leaks, or security flaws in the current codebase.

## Goal

The ultimate objective of this project is to achieve the highest possible degree of security. I am confident that this goal will be reached in due time, but it is not there yet.

## Reporting Vulnerabilities

Unlike many projects that require private disclosure, the current policy is as follows:

- You are free to report any found vulnerabilities publicly (via GitHub Issues).

- Since the project is in early development, public discussion of flaws helps improve the foundation.
