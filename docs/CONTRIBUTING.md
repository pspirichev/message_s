# Contributing

## Current Project Status

This project is currently in its <ins>**very early stages**</ins> of development. Because the core architecture is still evolving rapidly, I am not yet ready for large-scale contributions or new feature proposals.

## How to Help
### 1. Future Contributions

If you have great ideas for the project's development, I appreciate your enthusiasm! However, I ask that you hold onto your suggestions until project reaches at least a pre-alpha version. This ensures that your efforts won't be wasted on parts of the code that might be completely rewritten tomorrow.

### 2. Bug Fixes and Refactoring

If you've spotted a bug or have an improvement for the existing codebase:

- Feel free to submit a Pull Request with your changes.

- Please ensure your commits are concise and clearly describe what has been fixed or improved.

## Summary

I am building the foundation right now. Once the project stabilizes, I will open up more specific guidelines for community involvement. Thank you for your patience and for following the progress!
